# Problema "soma_matrizes"

Ler duas matrizes de números inteiros **A** e **B**, contendo **M**
linhas e **N** colunas cada (M e N máximo = 10). Em seguida, gerar uma
terceira matriz **C**, onde cada elemento é a **soma dos elementos
correspondentes** das matrizes A e B. Por fim, imprimir a matriz
resultante.

------------------------------------------------------------------------

## Exemplo

**Entrada:**

Quantas linhas vai ter cada matriz? 2\
Quantas colunas vai ter cada matriz? 3

Digite os valores da matriz A:\
Elemento \[0,0\]: 3\
Elemento \[0,1\]: 5\
Elemento \[0,2\]: 2\
Elemento \[1,0\]: 4\
Elemento \[1,1\]: 5\
Elemento \[1,2\]: 1

Digite os valores da matriz B:\
Elemento \[0,0\]: 2\
Elemento \[0,1\]: 4\
Elemento \[0,2\]: 5\
Elemento \[1,0\]: 1\
Elemento \[1,1\]: 8\
Elemento \[1,2\]: 8

**Saída:**

MATRIZ SOMA:

    5 9 7
    5 13 9
