# Negativos na Matriz

Ler dois números inteiros **M** e **N** (máximo = 10) e, em seguida, ler uma matriz **M x N** contendo números inteiros. O programa deve mostrar na tela **apenas os valores negativos** presentes na matriz.

---

## Exemplo

### Entrada

```
Qual a quantidade de linhas da matriz? 2
Qual a quantidade de colunas da matriz? 3
Elemento [0,0]: 12
Elemento [0,1]: -8
Elemento [0,2]: 5
Elemento [1,0]: -13
Elemento [1,1]: 10
Elemento [1,2]: -6
```

### Saída

```
VALORES NEGATIVOS:
-8
-13
-6
```
